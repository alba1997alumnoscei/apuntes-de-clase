box model con html, nada de css

Tienen 4 elementos: - content (contenido) - padding (relleno ) - border (border) -margin (margen)

1) content: es el espacio donde se muestra nuestro contenido, como por ejemplo puede ser texto, imágenes, videos... CSS: height, width, min-height, min-width, max-height, max-width espacio interno


2) padding: es el espacio entre el contenido y el borde del elemento. Podemos definir este espacio para los cuatro lados de la caja (arriba, derecha, abajo, izquierda) CSS: padding, padding-top, padding-right, padding-bottom, padding-left espacio interno espacio entre contenido/relleno y borde

3)border: es la linea que rodea el contenido y el relleno. Puede tener un ancho, estilo y color. CSS: boder, border-style, border-color, border-width esapcio interno línea que rodea la caja

4)margin: espacio entre el borde del elemento/caja y los otros elementos de nuestra web. Se puede definir para los cuatro lados de la caja. (arriba, derecha, abajo, izquierda) CSS: margin, margin-top, margin-right, margin-bottom espacio externo espacio que hay entre cajas.

Cajas en bloque:

cajas de div: ocupa todo el ancho ej: los parrafos son cajas de bloque.

Cajas inline: cajas de span: ocupa espacio que necesita. ESta se usa para texto sobretodo. las imagenes son cajas de línea. img : se comporta como una inline pero me deja ponerle un width y un source
